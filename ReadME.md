# Learning - C

To be a really good mathematician, ML reseacher or even quant it is important to know how to code at low levels.   
Usually this is done through C or C++ so I am going to learn C. So exciting!!  
I see so many Youtubers using C and C++ to make amamzing things so I thought I would learn it too!!  

This will be facilitated by the book  "Modern C" by Jens Gustedt. 

## HOW IT'S MADE 

### LEVEL0 - ENCOUNTER   
This familiarises me with key terminology needed to navigate C such as such as key words, functions, and overal formulation of C programs. This section was split up into the parts:
1) Useful terms
2) Structure of programs
    - declarations of identfiers 
    - defintions of identifiers 
    - statements that allow us to manipulate and use them 
3) Summary of important topics 

In this level we created the short C programs `main.c` and `getting_started.c` to explore things like block scope, separators, initialisation and much more.  

### LEVEL 1 - ACQUAINTANCE   

This level begins with pivotal control statements namely: for, do, while, switch, and if. 

I made two new files called `scratch_1.c` and `scratch_2.c` that allowed me to play with conditional exections were I learned:
    - layout of block and scope structure in C 
    - how to use variables for True and False conditional executions
